About Me

Welcome to my website! If you can't tell from the title of the website, my name is Kevin Wong, and I'm a student currently working towards a B.S. in Computer Science at the University of California, San Diego.

Projects

Shallus-Bot

A bot for the messaging application Discord, which is built off of the "Red Discord-Bot" by TwentySix on GitHub.
I implemented a system similar to a trading card system in which users can draw, collect, and trade images pulled from a database using its REST API. These images are stored in lists which are assigned to each user, and are stored in JSON files to disk in order to allow them to persist.

NyaBot

Another bot for Discord. This bot is my own bot, and is primarily a soundboard, playing sound files on command through the application into a voice channel.
It makes extensive use of the Discord api as well as uses mechanisms such as a pseudo-implementation of semaphores in order to make the asynchronous nature of its commands more scalable.
